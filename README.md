# TTClickCell Guide Repository

이 저장소는 TTSoft의 AI 기반 Excel 자동화 소프트웨어 **TTClickCell**의 기술 문서 및 내부 개발 가이드를 포함합니다.

- [📘 Comprehensive Guide (v1.1)](./Comprehensive_Guide.md):  
  프로젝트 전체 구조, 시스템 아키텍처, AI 연동 및 배포 가이드 포함.

- [🧠 GEMINI.md](./docs/GEMINI.md):  
  TTClickCell 프로젝트의 헌장 문서로, 코드 보존·수정 원칙, Gemini CLI 연동 규칙,  
  개발 환경(WSL·Nuitka·MSIX) 및 배포 지침을 정의합니다.

- [⚙️ GEMINI_CLI_PROTOCOL.md](./docs/GEMINI_CLI_PROTOCOL.md):  
  Gemini CLI 전용 운용 프로토콜로, 자동 수정·백업·테스트 포맷 등 실제 실행 절차를 명시합니다.  
  `GEMINI.md`와 함께 항상 참조되어야 합니다.

> 본 문서는 SK㈜ AI ERP팀 지원 시 제출된 기술 포트폴리오의 일부로 작성되었습니다.
